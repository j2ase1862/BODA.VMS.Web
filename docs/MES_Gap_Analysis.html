9 잔여 격차 보강

✓ G3 잔여: Operator Logging UI + Kiosk

VMS 클라이언트 수정 없이 키오스크 단말로 작업자 인증. 검사 결과 수신 시 라인의 현재 작업자에게 자동 귀속.

Entities: Operator (사번 unique, PinHash=BCrypt) · OperatorSession (라인별 현재 작업자)
Services: OperatorService · OperatorSessionService
API:
  • /api/kiosk/login · logout · current · line (Anonymous)
  • /api/operators CRUD + /sessions 이력 (Authorized)
Pages: /operators (Admin) · /kiosk/{ClientIndex} (Anonymous EmptyLayout)
자동 매칭: request.OperatorId ?? ResolveCurrentOperatorIdAsync(clientId). 다른 작업자 로그인 시 기존 세션 EndReason=Auto로 자동 종료 (교대).
✓ G16: 예방 보전 + 설비 신뢰성

SEMI E10 RAM 표준 — 계획 정지(PM) 관리 + 신뢰성 통계.

Entities: MaintenanceSchedule (ClientId nullable=공통) · MaintenanceRecord
Services: MaintenanceService · ReliabilityService
API:
  • /api/maintenance/schedules CRUD + /due + /{id}/perform + /records
  • /api/reliability/calculate (라인별 MTBF/MTTR)
Pages: /maintenance (PM 일정/임박/이력 3탭) · /reliability (MTBF/MTTR 카드)
MTBF = Σ(Running 시간) / 고장 횟수 · MTTR = Σ(Down 시간) / 고장 횟수 · 가용성 = MTBF / (MTBF + MTTR). EquipmentStatusLog의 Down 전이를 기간 내에서 카운트. 가용성 등급: ≥95% Success / ≥85% Info / ≥70% Warning / <70% Error.

  • 품질 관리: 파레토 · SPC · 불량 코드 · 교대 리포트 · PDF 리포트
